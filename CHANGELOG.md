# Changelog

All notable changes to Inline Feedback are documented here.

## 1.0.0

- Initial public release.
- Add inline feedback annotations for selected text in Markdown notes.
- Store structured feedback in `<note>.feedback.json`.
- Export readable feedback summaries to `<note>.feedback_export.md`.
- Append annotations to a local `feedback_log.md` review log.
- Add a feedback sidebar with navigation and deletion.
- Add Knowledge Cards for saving reusable excerpts, keywords, categories, and source links.
- Add desktop-first public documentation for GitHub and BRAT installation.
