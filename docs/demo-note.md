# Demo Note: AI Product Review

This sample note is safe to use for screenshots, testing, and demo recordings.

## Draft

AI copilots are now common in writing tools, but many review workflows still rely on scattered chat messages. The gap is not model quality alone. The gap is that writers need a precise way to show which sentence should change and why.

Shared vaults make this more important. One person may review a draft on a laptop, while another machine or writing agent performs the revision later. A plain Markdown file is easy to sync, but it needs a lightweight comment layer that stays next to the source note.

Inline feedback solves this by storing each annotation in a companion JSON file. The source note remains clean, while the feedback stays structured enough for a human reviewer or AI assistant to process.

## Suggested Demo Annotations

- Select "scattered chat messages" and add: "Give a concrete example of this pain point."
- Select "The gap is not model quality alone" and add: "Make this claim less absolute."
- Select the second paragraph and save it as a Knowledge Card with category `Evidence`.
